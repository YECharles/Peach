Class UnixDebugger

source code


Unix vtrace monitor.  This debugger monitor uses the gdb
debugger via pygdb wrapper.  Tested under Linux and OS X.

        * Collect core files
        * User mode debugging
        * Capturing stack trace, registers, etc
        * Symbols is available
