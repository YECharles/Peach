HandleDataContainerChildren(self, node, parent, errorOnUnknown=True)

source code 

Handle parsing conatiner children. This method will handle children of DataElement types for containers like Block, Choice, and Template.

Can be used by Custom types to create Custom container types.

Parameters:
  • node (XML Element) - Current XML Node being handled
  • parent (ElementWithChildren) - Parent of this DataElement
  • errorOnUnknonw - Should we throw an error on unexpected child node (default True)
  • errorOnUnknown (Boolean)

_locateDefaultMutators(self, obj=None)

source code  

Look for a default set of mutators.  We will follow this
search pattern:

1. Look at our self (context) level
2. Look at our imported namespaces
3. Recerse into namespaces (sub namespaces, etc)

This means a <Mutators> element in the top level XML file will
get precidence over the defaults.xml file which is included into
a namepsace.
