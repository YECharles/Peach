read(self, size=None)

source code 

Read from current position. If size isn't specified, read rest of stream.

Read will shift the current position.

write(self, data, name=None)

source code 

Write a block of data at current position. Stream will expand if needed to support the written data. Otherwise it will overright the existing data.

Parameters:
  • data (string) - Data to write
  • name (string) - Name to store position under [optional]
