source code 

A Peach reporting server client. Communicates to server portion (twistedserver.py) via XML-RPC. Many clients can send results to a single server for collection.

NOTE: DNS lookup is *slow*, use and IP address even for localhost!
Parameters:
  • url (string) - URL of Peach server ("http://127.0.0.1:8000")
  • client (string) - Name of client (defaults to hostname)
