Class Block

source code

Block is a set of Generators in a specific order who's values are combined into a block of data. The BlockSize generator can be used to output the size of the block or other generators.

Example:

>>> gen = Block([
...  Static('Hello'),
...  Static(' '),
...  Static('World')
...  ])
>>> print gen.getValue()
Hello World
Notes:
  • This type of block will not call .next() on the generators it contains. See Block2 or Block3 for that.
  • Do not use with DictionaryList generator. Will cause infinitlooping since the .next() is never called on sub generators.

See Also: MultiBlock, Block2, Block3, BlockSize, MultiBlockCount
