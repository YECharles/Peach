Class BadNumbers

source code

Generate numbers that may trigger integer overflows for both signed and unsigned numbers. Under the hood this generator performs a NumbersVariance on the boundry numbers for:

  • int8 (0, -128, 127)
  • unsigned int8 (255)
  • int16 (-32768, 32767)
  • unsigned int16 (65535)
  • int32 (-2147483648, 2147483647)
  • unsigned int32 (4294967295)
  • int64 (-9223372036854775808, 9223372036854775807)
  • unsigned int64 (18446744073709551615)

See Also: BadNumbers16, NumbersVariance, BadUnsignedNumbers, BadPositiveNumbers
