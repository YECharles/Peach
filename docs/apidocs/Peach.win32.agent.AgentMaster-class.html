Class AgentMaster

source code

_AgentCommon --+
               |
              AgentMaster
The AgentMaster controls the remote Agent. Typically the master is what generates the tests and the agent detects issues, or causes a client to connect, etc. This agent code uses named pipes and therefor is only usefull on Windows systems.

Notes:
  • pywin32 is required
  • Message size is a max of about 1MB. This can be changed by editing the AgentMaster.__init__ method and changing the two numerical values to something higher.
