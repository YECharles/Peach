Class MultiBlockCount

source code

Generates the number of occurances of a MultiBlock generator.

Example:

>>> block = MultiBlock([ Static('12345') ], 0, 100, Static(1))
>>> blockCount = MultiBlockCount( block )
>>> print blockCount.getValue()
1

See Also: MultiBlock
