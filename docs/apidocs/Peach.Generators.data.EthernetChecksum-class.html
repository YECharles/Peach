Class EthernetChecksum

source code

Will generate the Ethernet checksum of Block or another Generator. EthernetChecksum can can detect recursive calls and provides a default based on ICMP RFC.

Produced value is 16 bits.

Example:

>>> block = Block([ Static('12345') ])
>>> icmpChecksum = IcmpChecksum( block )
>>> print icmpChecksum.getValue()
5
